정량펌프란? —
연동식 정량·디스펜싱 선택 가이드

"정량펌프"는 정해진 양을 정밀하게 반복 투입하는 펌프의 통칭입니다. LeadFluid에서는 연동식(BQ80S·BT-F)이 정량·분주를 담당합니다. 어떤 모델이 어떤 정량 작업에 맞는지 정리했습니다.

정량펌프는 정해진 양의 유체를 정밀하게 반복 투입·분주하는 펌프의 통칭입니다. 실험실에서 무오염·저유량 정량이 필요할 때는 유체가 튜브 안에서만 흐르는 연동식 정량펌프를 주로 씁니다.

LeadFluid 정량은 "연동식"으로 구현됩니다

LeadFluid에는 별도의 다이어프램 정량펌프 라인이 없습니다. 대신 연동펌프의 정량 주입 기능으로 정량·디스펜싱을 구현합니다. 유체가 튜브 내부로만 흐르므로 정량을 하면서도 무오염·멸균이 쉽고, 튜브 교체만으로 다른 시약으로 전환할 수 있습니다. 기어식 정량이 필요한 특수 용도는 별도 상담이 필요합니다.

어디에 쓰나

  • pH 제어 — 센서 신호에 따라 산·염기를 정량 투입해 pH 유지. → 환경·수처리 셋업
  • 시약·영양 정량 공급 — 반응기·배양기에 정해진 양을 주기적으로 투입.
  • 분주·디스펜싱 — 정해진 부피를 반복해 여러 용기에 나눠 담기.

유량 범위는 튜브 규격·헤드 조합의 상·하한입니다. 분주 정확도는 튜브 캘리브레이션과 제어 방식에 따라 달라집니다.

정량과 무오염을 동시에

연동식 정량은 유체가 튜브 안에서만 흐르기 때문에, 정해진 양을 반복 투입하면서도 시료 오염이 없습니다. 연동펌프 전반의 원리·모델은 연동펌프 가이드에서 확인하세요.

실험셋업연구소 제어 소프트웨어로 정량 자동화·재현

실험셋업연구소는 LeadFluid 정품 공급·국내 직접 A/S와 함께, 펌프 제어 프로그램을 직접 만들어 구매 고객에게 무상 제공합니다. 정량 스케줄·pH 연동 투입·분주 반복을 자동화하고 CSV 로그로 재현 가능한 정량 데이터를 남길 수 있습니다. 하드웨어는 국내에서 직접 진단·수리하며 실험셋업연구소 구매 고객에게 3년 무상보증을 제공합니다.
