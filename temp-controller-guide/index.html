자주 묻는 질문 (FAQ)

버튼을 눌러도 반응이 없어요

Key 잠금(LOCK)이 설정된 경우입니다. 제어그룹(G.CTL)의 LOCK 파라메터를 해제하세요.

오토튜닝(AT)이 실행되지 않아요

오토튜닝은 RUN 상태에서만 실행됩니다. 운전을 먼저 시작하고 PV 입력(센서)이 정상인지 확인한 뒤, 오토튜닝 그룹(G.AT)에서 AT=ON으로 설정하세요. 입력 단선 시 자동 중단됩니다.

프로그램이 반복되지 않아요

RPT(횟수)·RST(시작)·REN(종료)을 모두 설정했는지, RST ≤ REN 조건을 만족하는지 확인하세요. 패턴 전체 반복은 종료동작(LC)을 자기 패턴(PTN1/PTN2)으로 지정합니다.

운전 시작 시 현재 온도부터 시작해요

운전 시작 조건(STC) 기본값이 PV이기 때문입니다. 설정한 시작점부터 시작하려면 STC=SSP로 변경하고 SSP 값을 설정하세요.

세그먼트가 넘어가지 않고 멈춰 있어요

대기동작(Wait) 때문일 수 있습니다. PV가 대기범위(W.ZON)에 들어올 때까지 대기하며, 대기시간(W.TM) 경과 시 자동 진행됩니다. 원치 않으면 W.ZON=OFF로 설정하세요.

RS485 통신이 안 돼요

통신 옵션 장착 여부, A/B 배선, COM.P·BAUD·PRTY·S.BIT·D.LEN이 상위 프로그램과 일치하는지, ADDR 중복이 없는지 확인하세요.

텍스트로 보는 사용법 — 프로그램 운전·반복·RS485 통신 (전체 절차)

삼원테크놀로지 NOVA500E 시리즈(SP590 등) 프로그램형 온도컨트롤러의 기본 운전 절차입니다. 실제 기기에서 SET 길게 누르기는 3초 이상입니다. (사용설명서 Ⅰ-26~38 참고)

온도 스케줄 설정

  1. 운전화면에서 SET 버튼을 3초 이상 길게 눌러 설정화면으로 이동
  2. SET로 프로그램 그룹(PTNO) 진입 — 시간단위(TMU, 시.분/분.초) 확인
  3. 세그먼트1 목표온도(1.SP1): ◀로 자릿수 이동, ▲▼로 값 변경 후 SET 저장 (예: 80.0℃)
  4. 세그먼트1 운전시간(1.TM1): 예 00.30(30분) — 승온속도 = (80−25)℃ ÷ 30분 ≈ 1.8℃/분. 시간이 짧을수록 빠르게 승온
  5. 유지 구간: 1.SP2에 같은 온도(80.0), 1.TM2에 유지시간(예: 01.00 = 1시간) 입력
  6. 하강 구간: 1.SP3에 낮은 온도(예: 25.0), 1.TM3에 하강시간 입력. 세그먼트는 패턴당 최대 15개
  7. SET 3초 이상 → 운전화면 복귀

운전 시작·정지

  1. 패턴1 운전 시작: ▼/P1 버튼 1초 누름 (패턴2는 ▲/P2)
  2. 운전 정지: ◀/RST 버튼 1초 누름 → STOP(RESET) 상태

구간 반복 (사이클 시험)

  1. 반복횟수(1.RPT): 1~999회, 0 = 무한반복
  2. 반복 종료 세그먼트(1.REN)와 반복 시작 세그먼트(1.RST) 설정 — 조건: RST ≤ REN
  3. 예: RPT=3, RST=1, REN=2 → 세그먼트 1~2를 3회 반복 후 다음 진행

패턴 종료 시 동작 (LINK CODE, 1.LC)

RST: 운전 정지(기본값) · HOLD: 마지막 SP 온도 유지(RST 키로 해제) · PTN1/PTN2: 해당 패턴으로 연결 운전 — 자기 패턴을 지정하면 무한반복 운전이 됩니다. (사용설명서 Ⅰ-37, 39 참고)

RS485 통신 설정 (G.COM)

  1. 설정화면에서 ▲로 그룹 이동, 제어그룹(G.CTL)을 지나면 암호(PWD) 화면 — 초기값 0
  2. 통신그룹(G.COM) 진입 (통신 옵션 장착 시 표시)
  3. 통신 프로토콜(COM.P): PCC0/PCC1(초기값)/MBS.A(MODBUS ASCII)/MBS.R(MODBUS RTU) 등
  4. 통신 속도(BAUD): 9600~115.2K (초기값 38.4K)
  5. 패리티(PRTY): NONE/EVEN/ODD · 정지비트(S.BIT): 1/2 · 데이터길이(D.LEN): 7/8 (MODBUS 시 미표시)
  6. 통신 주소(ADDR): 1~99, 주소 중복 금지, 최대 31대 연결 (사용설명서 Ⅰ-92~93 참고)
