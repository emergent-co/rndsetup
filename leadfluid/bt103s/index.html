리드플루이드 BT103S 분주형 연동펌프 (LeadFluid BT103S)
— 반복 분주, 편차 없이 무인으로

최종 업데이트 2026년 7월 · 스펙 출처: LeadFluid 2025 공식 카탈로그

밤새 손으로 시약을 나눠 담다 마지막에서 양이 어긋나 실험을 다시 돌린 적 있으신가요? 반복 분주에서 편차가 생기면 결과 전체가 흔들립니다. BT103S는 정해진 부피를 정해진 간격으로 자동 반복 분배해, 사람이 지키고 있지 않아도 같은 양을 재현합니다.

한 줄 요약 — BT103S는 리드플루이드 분주형 지능형 연동펌프로, 유량 0.00011~500 mL/min, 속도 정확도 ±0.2%. 디스플레이에서 분주 부피·간격·반복을 설정해 정량을 반복 분배하고, 필요하면 RS485로 PC·자동화까지 확장합니다.

BT103S는 어떤 펌프인가 — 분주(디스펜싱) 중심

리드플루이드(LeadFluid) BT103S는 분주(디스펜싱) 기능을 갖춘 지능형 연동(페리스탈틱) 펌프입니다. 그냥 흘려보내는 이송이 아니라, 정해진 부피를 정해진 간격으로 반복 분배하는 데 특화돼 있습니다. 디스플레이에서 1회 분주 부피·간격·반복 횟수(무한 루프 포함)를 설정하면, 사람이 없어도 같은 양을 재현합니다.

튜브만 교체하면 유체가 펌프 본체에 닿지 않아 오염에 민감한 시약·배지 분주에 적합하고, 실제 유량은 펌프헤드·튜브 규격 조합에 따릅니다. 시약·시료를 여러 용기에 정량으로 나눠 담기, 일정 간격 반복 주입이 주 용도입니다.

우리 분주 조건에 BT103S가 맞을까요?
채널 수·1회 분주량·튜브만 남기면 구성과 실측 유량을 제안드립니다.

분주 셋업, 3단계로

1
펌프헤드·튜브 선택 — 분주량 범위에 맞춰 DG/DS·YZ15·YT25 중 헤드를 고르고 실리콘·PharMed 등 튜브 규격을 정합니다. (유량은 헤드·튜브 조합으로 결정)
2
분주 부피·간격·반복 설정 — 디스플레이에서 1회 분주 부피, 분주 간격, 반복 횟수(무한 루프 포함)를 입력합니다.
3
실측 보정 — 저울로 실제 토출량을 재서 보정계수를 잡으면 설정값과 실제가 일치합니다. 유량이 설정값과 다를 땐 원인·해결법을 참고하세요.

분주 셋업에서 흔한 실수 3가지 (유지보수 팁)

1. 헤드·튜브를 유량에 안 맞게 고름 — 목표 유량이 헤드 범위의 끝단에 걸리면 저유량에서 편차가 커집니다. 위 표에서 목표 유량이 중간에 오는 헤드를 고르세요.

2. 튜브 노후 방치 — 연동펌프는 튜브가 소모품입니다. 눌린 자국·경화가 생기면 유량이 변합니다. 정기 교체가 반복 정확도를 지킵니다. 유량이 설정값과 어긋나면 원인·해결법부터 확인하세요.

3. 수동 의존으로 편차·야간 공백 — 반복 분주를 손으로 하면 개인차·피로로 편차가 납니다. BT103S는 부피·간격을 고정해 무인으로 재현합니다.

무인 반복 분주가 만드는 부수 효과

야간·주말 확보 — 사람이 지키지 않아도 정해진 양을 반복 분배하니, 장시간 분주를 밤사이 돌리고 낮 시간은 실험 설계에 씁니다.

시약·시료 낭비 감소 — 편차가 줄면 재실험이 줄고, 필요한 양만 정확히 분배해 값비싼 시약 손실을 줄입니다.

재현 로그 확보 — 제어 소프트웨어로 언제 얼마를 분배했는지 기록해 논문 부록·데이터 검증에 그대로 씁니다.

분주 셋업, 이렇게 도와드립니다

BT103S로 분주 셋업을 계획 중이시라면 채널 수·1회 분주량·튜브·반복 주기만 문의로 남겨주세요 — 맞는 펌프헤드·튜브 구성과 실측 유량을 제안드립니다. 실험셋업연구소가 실제로 구성한 도입 셋업은 펌프 셋업 사례에서 볼 수 있습니다.

자동화·PLC·PC 제어가 필요하다면 (심화)

대부분의 분주는 위 설정만으로 충분합니다. PLC·센서 연동이나 PC 자동 제어가 필요한 경우만 아래를 보세요.

BT103S는 RS485·MODBUS와 DB15 외부제어를 지원합니다. Start/Stop·정역은 무전압 접점(5~24V), 속도는 아날로그 0~5V·0~10V·4~20mA로 제어돼 PLC·센서 연동에 유리합니다. 실험셋업연구소 구매 시 PC 제어 소프트웨어로 유량·분주를 스케줄 운전하고 운전 로그(CSV)로 재현합니다. 내 장비가 제어되는지는 PC 제어 확인 →, 통합 자동화는 통합 제어 소프트웨어 →.

Python 제어 예제 보기 (RS485·MODBUS)

범용 라이브러리로 유량·구동을 제어합니다. (레지스터 주소·바이트오더는 BT103S 매뉴얼 Register Map 참조)

import minimalmodbus, time
# 레지스터 주소·바이트오더는 BT103S 매뉴얼(Register Map) 참조
REG_FLOW, REG_RUN = 0x0000, 0x0000        # ← 매뉴얼 값으로 대체

pump = minimalmodbus.Instrument('COM3', 1)   # 포트, 슬레이브 주소
pump.serial.baudrate = 9600
pump.serial.parity = minimalmodbus.serial.PARITY_EVEN   # 예: 8E1

pump.write_register(REG_RUN, 1, functioncode=6)   # Start
pump.write_float(REG_FLOW, 5.0)                   # 유량 5.0 mL/min
time.sleep(300)                                   # 5분 유지
pump.write_register(REG_RUN, 0, functioncode=6)   # Stop

왜 실험셋업연구소에서 사는가 — 3원칙

원칙 1 · 정품만 공급 — 리드플루이드 정품만 취급하고, 출처가 불분명한 병행·리퍼 제품은 다루지 않습니다.

원칙 2 · 국내에서 직접 A/S — 고장 시 해외 반송 없이 국내에서 진단·수리합니다. 타사 구매품도 무료 진단합니다.

원칙 3 · 제어까지 책임 — 파는 데서 끝내지 않고 PC 제어 소프트웨어·유량 세팅·자동화 구성까지 함께 잡아 "실험이 실제로 돌아가게" 만듭니다.

정품·A/S·구매는?

실험셋업연구소는 리드플루이드 정품을 공급하고 국내에서 직접 A/S합니다. 구매 시 국내 직접 A/S·PC 제어 소프트웨어·3년 무상보증(구매 조건)이 적용되고, 타사 구매품도 무료 진단 후 수리합니다. 국내 A/S·정품·보증은 리드플루이드 브랜드 안내에서 확인하세요.

분주 부피·간격·튜브·유량 조건만 남기면 펌프·제어 구성까지 제안합니다.

실험 상담·견적 리드플루이드 전 모델

BT103S 자주 묻는 질문

리드플루이드 BT103S는 어떤 펌프인가요?
분주 기능을 갖춘 지능형 연동펌프로 유량 0.00011~500 mL/min, 회전수 0.1~100 rpm(정확도 ±0.2%). 분주 부피·시간·간격을 설정해 정량을 반복 분배합니다.
분주(디스펜싱) 기능은 어디에 쓰나요?
정해진 부피를 정해진 간격으로 반복 분배하는 정량 분주에 사용합니다. 시약·배지·시료 분배에 적합하며 부피·시간·간격·반복(무한 루프)을 설정합니다.
리드플루이드 BT103S 유량 범위가 어떻게 되나요?
헤드·튜브 조합에 따라 0.00011~500 mL/min입니다. 헤드별 상세 유량은 위 적용 펌프헤드·튜브별 유량 표를 참고하세요.
BT103S 가격은 얼마인가요?
펌프헤드·튜브·채널·수량 구성에 따라 달라져 견적으로 안내합니다. 조건 남기고 견적받기 또는 나비엠알오.
BT103S를 Modbus로 제어할 때 통신 설정은?
RS485·MODBUS 지원(예: 9600·8E1). 레지스터 주소·바이트오더는 BT103S 매뉴얼 Register Map을 따르며, PC 제어 확인에서 방법을 안내합니다.
BT103S에 맞는 튜브 규격은 무엇인가요?
헤드별로 다릅니다 — YZ15/YZ15T: 13#·14#·19#·16#·25#·17#·18#, YT25: 114#·116#·15#·24#·35#·36#, DG/DS: 내경 ID≤3.17mm.
리드플루이드 BT103S와 BT100S 차이는 뭔가요?
BT103S는 분주(부피·간격 반복)에 특화된 지능형 모델, BT100S는 기본 이송형입니다. 정량 반복 분배가 필요하면 BT103S가 맞습니다.
BT103S A/S는 국내에서 받을 수 있나요?
네. 실험셋업연구소가 국내에서 직접 진단·수리하며 구매 시 3년 무상보증(조건)이 적용됩니다. 타사 구매품도 무료 진단합니다.
