리드플루이드 CT3001F 마그네틱 기어펌프
— 무누설 연속 이송, CO₂ 포집 논문 사용

CT3001F는 리드플루이드(LeadFluid)의 마그네틱(자기구동) 기어펌프로, 축 실링이 없어 무누설로 유체를 저맥동·연속 이송합니다. 연속 CO₂ 포집 논문(Nature Communications 2024)에 사용됐고, 실험셋업연구소는 포집 공정 셋업 사례와 제어 소프트웨어·국내 직접 A/S를 함께 제공합니다.
구동 방식자기구동(마그네틱 커플링) — 축 밀봉부 없음, 무누설
유체 특성저맥동·일정 연속 유량 (기어펌프)
통신RS485 · Modbus RTU (지원 구성, 9600·8E1·슬레이브 1~247, 사양서 기준)
유량·양정·재질사양서 기준 — 유체 조건별 문의

CT3001F는 어떤 펌프인가?

리드플루이드(LeadFluid)의 자기구동(마그네틱 커플링) 기어펌프로, 자기력으로 기어를 돌려 동력을 전달하므로 회전축 밀봉부가 없어 누설·오염 위험이 낮은 것이 특징입니다. 기어펌프 특성상 맥동이 적고 일정한 연속 유량을 만들어, 흡수액 순환처럼 장시간 안정 이송이 필요한 공정에 적합합니다. 유량·양정·허용 점도·내약품 재질은 사양서 기준이며, 유체 조건별 선정은 문의로 안내합니다.

PC로 연속 운전을 제어할 수 있나?

통신 지원 구성에서 RS485·Modbus RTU로 회전수·유량·구동/정지를 원격 제어할 수 있습니다(9600·8E1, 슬레이브 1~247, 사양서 기준). 실험셋업연구소 구매 시 제어 소프트웨어로 장시간 연속 운전을 스케줄링하고 상태를 모니터링합니다. 관련: PC 제어 확인 · 소프트웨어 제어.

실제 어떤 공정·연구에 쓰였나?

논문 사용 — 연속 CO₂ 포집 연구(Nature Communications 2024)에서 흡수액 등 유체의 연속 순환·이송에 CT3001F가 사용됐습니다(서지·DOI는 사례 페이지 참조).

"소량 정밀 주입"이 아니라 "무누설·연속 순환"이 필요할 때 고르는 모델입니다.

정품·A/S·구매는?

실험셋업연구소는 리드플루이드 정품을 공급하고 국내에서 직접 A/S합니다. 구매 시 국내 직접 A/S·제어 소프트웨어·3년 무상보증(구매 조건)이 적용됩니다. 인증서·수리 사례는 국내 A/S·정품·보증에서 확인하세요.

CT3001F 자주 묻는 질문

리드플루이드 CT3001F는 어떤 펌프인가요?
자기구동 기어펌프로 축 밀봉부가 없어 무누설. 저맥동·일정 연속 유량이라 흡수액 순환 등 장시간 안정 이송에 적합. 유량·재질은 사양서 기준.
PC로 제어할 수 있나요?
통신 지원 구성에서 RS485·Modbus RTU로 회전수·유량 제어(9600·8E1·슬레이브 1~247, 사양서 기준). 구매 시 제어 SW로 연속 운전 스케줄링. 제어 확인.
실제 어떤 공정에 쓰였나요?
연속 CO₂ 포집 논문(Nature Communications 2024)에 흡수액 순환으로 사용. 사례 페이지.
정품·A/S·구매는?
리드플루이드 정품 공급·국내 직접 A/S, 구매 시 3년 무상보증. 나비엠알오 구매.
