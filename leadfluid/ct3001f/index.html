리드플루이드 CT3001F PEEK 기어펌프 (LeadFluid CT3001F)
— 저맥동 연속 이송

CT3001F는 리드플루이드(LeadFluid)의 PEEK 기어펌프로, 내약품·내마모성이 좋은 PEEK 기어로 유체를 저맥동·일정하게 연속 이송합니다. 유량은 펌프헤드에 따라 15~2700 mL/min. 실험셋업연구소는 제어 소프트웨어·국내 직접 A/S를 함께 제공합니다.

CT3001F는 어떤 펌프인가?

리드플루이드(LeadFluid)의 PEEK 기어펌프로, 맞물린 기어가 유체를 밀어내는 방식이라 맥동이 적고 일정한 연속 유량을 만듭니다. 기어를 내약품·내마모성이 우수한 PEEK로 만들어, 용매·점성 유체를 오래 돌려도 마모·손상 위험이 낮습니다. 흡수액 순환처럼 장시간 안정 이송이 필요한 공정에 적합합니다. 유량은 펌프헤드에 따라 15~2700 mL/min이며, 양정·허용 점도·내약품 재질은 사양서 기준이고 유체 조건별 선정은 문의로 안내합니다.

PC로 연속 운전을 제어할 수 있나?

통신 지원 구성에서 RS485·Modbus RTU로 회전수·유량·구동/정지를 원격 제어할 수 있습니다(9600·8E1, 슬레이브 1~247, 사양서 기준). 실험셋업연구소 구매 시 제어 소프트웨어로 장시간 연속 운전을 스케줄링하고 상태를 모니터링합니다. 관련: PC 제어 확인 · 소프트웨어 제어.

Python 예제 — CT3001F 연속 운전을 코드로 제어하려면?

통신 지원 구성에서 RS485·Modbus RTU로 회전수·유량·구동을 제어합니다. (정확한 레지스터 주소·바이트오더는 CT3001F 매뉴얼을 따르세요.)

import minimalmodbus, time
# 레지스터 주소·바이트오더는 CT3001F 매뉴얼(Register Map) 참조
REG_FLOW, REG_RUN = 0x0000, 0x0000        # ← 매뉴얼 값으로 대체

pump = minimalmodbus.Instrument('COM3', 1)   # 포트, 슬레이브 주소(1~247)
pump.serial.baudrate = 9600
pump.serial.parity = minimalmodbus.serial.PARITY_EVEN   # 8E1

pump.write_register(REG_RUN, 1, functioncode=6)   # 연속 운전 시작
pump.write_float(REG_FLOW, 500.0)                 # 유량 500 mL/min
time.sleep(86400)                                 # 24시간 무인 연속 순환
pump.write_register(REG_RUN, 0, functioncode=6)   # Stop

어떤 공정에 적합한가?

적합 용도 — PEEK 기어로 저맥동·일정하게 연속 순환·이송하는 공정(흡수액 순환·용매 이송·연속 반응 공급)에 적합합니다.

"소량 정밀 주입"이 아니라 "일정한 연속 순환·이송"이 필요할 때 고르는 모델입니다.

정품·A/S·구매는?

실험셋업연구소는 리드플루이드 정품을 공급하고 국내에서 직접 A/S합니다. 구매 시 국내 직접 A/S·제어 소프트웨어·3년 무상보증(구매 조건)이 적용됩니다. 인증서·수리 사례는 국내 A/S·정품·보증에서 확인하세요.
