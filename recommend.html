Q 선택 전, 무엇을 먼저 정리해두어야 하나요? +

본체 모델명만 보고 결정하기는 어렵습니다. 셀렙에 문의 주실 때 아래 4가지가 정리돼 있으면 제품군과 구성품을 빠르게 좁힐 수 있습니다.

1

사용 목적

연속 이송인지, 반복 분주인지, μL 단위 미량 주입인지부터 구분합니다. 대용량 공급·장비 내장·방폭 환경처럼 특수 조건이 있다면 처음부터 별도 라인으로 분리해 보는 게 좋습니다.

2

유량 범위

최소·최대 유량을 mL/min 또는 L/min으로 함께 잡아두면 본체·헤드·튜브 조합을 결정하기 쉽습니다. 한쪽만 정해두면 헤드 호환에서 막히는 경우가 많습니다.

3

액체 특성

점도·부식성·휘발성·인화성·침전물·멸균 필요 여부를 확인합니다. 액체가 튜브와 직접 닿는 페리스탈틱 방식에서는 튜브 재질 호환성이 모델 선택보다 먼저 검토되어야 합니다.

4

제어 방식·설치 환경

수동 다이얼이면 충분한지, 외부 신호·PLC 연동이 필요한지, 또는 장비 내부에 들어가야 하는지에 따라 본체급이 달라집니다. 방폭 구역은 별도 인증된 라인으로 검토합니다.

Q 정량펌프에는 어떤 종류가 있나요? +

크게 페리스탈틱 계열(롤러로 튜브를 압착) · 시린지 · 기어 · 분주 자동화로 나뉩니다. 같은 페리스탈틱이라도 RPM 기준이냐 유량 기준이냐, 1채널이냐 다중채널이냐에 따라 운전 감이 꽤 달라집니다.

기본 이송

스피드 정량펌프

드라이버 RPM을 직접 조절해 액체를 보내는 가장 기본형. 정확한 1회 분주량보다 안정적인 연속 이송이 우선일 때.

이런 경우배지·완충액·세척액 일반 이송 / 수조·반응조 보조 라인 / 여러 용도 겸용 펌프

유량 기준

유량조절 정량펌프

RPM이 아니라 mL/min 단위로 운전 조건을 잡습니다. 작업자별 편차를 줄이고 표준화가 필요한 라인에 적합합니다.

이런 경우세포배양·발효 일정 공급 / 분석 전처리 라인 / 운전 조건 표준화

반복 분주

디스펜스 정량펌프

1회 분주량·반복 횟수·간격을 미리 입력해 반복 작업하는 타입. 흘려보내는 작업과 다른 원리입니다.

이런 경우바이알·튜브 소분 / QC 샘플·키트 충진 / 피펫팅 시간·편차 감소

복수 라인

다중채널 디스펜스

여러 라인을 같은 조건으로 동시에 운전. 채널마다 다른 유량을 원할 경우는 독립 제어 가능 모델인지 별도 확인이 필요합니다.

이런 경우플레이트·바이알 동시 분주 / 병렬 반응 / 자동화 라인 처리량 향상

분주 자동화

자동분주기

분주 작업 자체를 자동화한 형태. 분주 대상 용기·횟수에 따라 본체 구성이 달라집니다.

이런 경우시약병·키트 반복 충진 / 작업자 피로 감소 / 소량 배치 생산·전처리

미량 주입

시린지펌프

시린지를 직접 밀어 μL 단위로 정밀 주입. 튜브를 계속 회전시키는 방식이 아닌, 정해진 부피만 정확히 보내는 용도.

이런 경우마이크로플루이딕스·전기화학 / 분석기 시료 주입 / 약물 반응 재현성

점성·압력

기어펌프

기어 회전으로 액체를 밀어내는 방식. 점도가 높거나 자연압·배관 압력 때문에 페리스탈틱만으로 토출이 흔들릴 때 같이 검토합니다.

이런 경우오일·수지·코팅액 / 자연압 환경 / 정량 도포·소량 충진

안전 체크. 인화성 액체를 쓸 때는 펌프 방식 외에 방폭 등급·재질 호환성·현장 안전 기준도 같이 보셔야 합니다.
장비 내장

OEM 정량펌프

완성 장비 안에 들어가는 모듈형. 사양뿐 아니라 장착 공간·전원·통신 방식까지 같이 맞춰야 해서 초기 검토 항목이 더 많습니다.

이런 경우분석·세척·전처리기 내장 / PLC 연동 / 장비 개발 단계

대량 이송

대용량 정량펌프

실험실용 소형 펌프 한계를 넘는 처리량이 필요할 때. 헤드·튜브 내구성, 장시간 운전 조건이 핵심입니다.

이런 경우파일럿·배치 공정 / 폐수처리·약품 주입 / 식품·화장품 원료

안전 환경

방폭 정량펌프

인화·휘발성 액체 또는 방폭 구역에서 사용하는 인증 라인. 액체 성질뿐 아니라 설치 구역·방폭 등급을 함께 확인합니다.

이런 경우알코올·용매 취급 화학·석유화학 / 방폭 구역 설치 / 위험물 이송

Q 정량펌프는 무엇으로 구성되나요? +

본체 하나만 사면 끝이 아닙니다. 같은 본체라도 어떤 헤드·튜브를 끼우느냐에 따라 가능 유량과 적합한 액체가 달라집니다. 셀렙에서 구성을 안내드릴 때도 보통 다음 4가지를 묶어서 제안드립니다.

Driver

드라이버 (본체)

회전 속도·운전 조건을 제어합니다. 디지털 표시·분주 모드·외부 제어 입력 가능 여부가 본체급을 가르는 포인트.

Pump Head

펌프 헤드

실제로 튜브가 장착되어 동작하는 부품. 호환 튜브 규격·채널 수·유량 범위가 헤드에서 결정됩니다.

Tubing

튜브

액체와 직접 닿는 소모품. 화학 호환성·점도·멸균 가능 여부에 맞춰 재질을 고릅니다. 가장 자주 교체되는 부품.

Accessory

액세서리

커넥터·풋스위치·외부 제어 케이블 등. 운전 환경에 따라 필요 여부가 갈리는 보조 구성품입니다.

Q 헤드와 채널, 어떻게 다른가요? +

다중채널 펌프 문의에서 가장 자주 헷갈리는 부분입니다. 같은 본체라도 어떤 헤드를 다는지에 따라 사용 가능한 채널 수와 튜브 규격이 바뀝니다.

Head

헤드

튜브를 잡고 롤러로 압착해 이송하는 부품. 헤드 종류에 따라 들어가는 튜브 사이즈, 동시 채널 수, 유량 한계가 달라집니다.

Channel

채널

동시에 굴러가는 튜브 라인의 수. 1채널은 한 줄, 2채널은 두 줄, 다중채널은 여러 줄을 같은 회전 조건에서 함께 운전합니다.

채널별 독립 유량 제어가 가능한지 별도로 확인하세요

대부분의 다중채널 모델은 모터 하나가 여러 채널을 같이 돌리는 구조입니다. 채널마다 속도를 따로 설정하기보다 같은 회전에서 여러 라인을 동시에 운전한다고 보시는 게 정확합니다.

채널마다 다른 유량이 필요하면 채널별 독립 제어 모델인지 사양에서 확인이 필요합니다. 같은 RPM이어도 튜브 규격·장착 상태·점도에 따라 채널 간 실제 유량 편차는 항상 약간 발생합니다.

Q 튜브 번호와 유량표, 어떻게 봐야 하나요? +

튜브는 가격이 작아 보여도 실제 유량과 안정성에 가장 직접적으로 영향을 주는 부품입니다. 셀렙에 견적 문의 주실 때 튜브 사양이 빠져 있는 경우가 많아, 자주 헷갈리는 포인트만 정리했습니다.

Tube Size

튜브 번호 = 직경 숫자가 아닙니다

#13, #14, #16 같은 표기는 튜브 직경이 아닌 사이즈 코드입니다. 즉 #13이 내경 13mm를 뜻하지 않습니다.

실제 내경·외경·벽 두께는 사용 펌프헤드와 제조사 규격표에서 별도로 확인해야 합니다.

ID / OD

ID, OD, 벽 두께

ID는 내경(Inner), OD는 외경(Outer). 벽 두께를 알면 두 값은 서로 환산 가능합니다.

외경 = 내경 + 벽 두께 × 2

유량은 주로 내경에 좌우되고, 헤드 장착 가능 여부는 외경·벽 두께에 좌우됩니다.

Flow Rate

유량표는 물 기준 참고값

대부분의 유량표는 물처럼 점성이 낮은 액체로 측정한 값입니다. 실제 유량은 점도·온도·튜브 재질·압착 상태·흡입/토출 높이·배관 저항에 따라 달라집니다.

점성이 있는 액체는 같은 RPM·같은 튜브여도 토출량이 줄어드는 경우가 많습니다.

Calibration

정량성이 중요하면 보정 필수

유량표 값은 절대 수치가 아니라 기준 조건의 측정값입니다. 튜브 공차·장착·사용 시간·점도에 따라 편차가 생기기 마련입니다.

정확한 분주가 필요한 작업이면 사양표만 믿지 말고 실제 액체·튜브 조합으로 한 번 캘리브레이션해두는 걸 권장합니다.

실제 셋업이 궁금하시면

특정 응용에 어떤 튜브·유량을 쓰는지 논문 기반으로 정리한 펌프셋업 리뷰에서 케이스를 확인하실 수 있습니다.

Q 어떤 분야에서 사용되나요? +

액체를 일정하게 보내거나 나누어야 하는 거의 모든 산업에서 쓰입니다. 같은 분야 안에서도 액체와 유량 조건에 따라 적합한 제품군이 달라집니다.

바이오·생명공학

배지·시약·완충액 이송, 발효·세포배양 공정 일정 공급.

의료·분석

시료 주입, 약액 공급, 분석 장비 연동 등 정밀 제어 라인.

화학·화장품

시약·용매 혼합, 소량 충진, 점성 원료 이송.

식품·농업

액상 원료, 첨가물·향료, 양액·비료 자동 공급.

폐수처리·세제

약품 주입, 처리액 이송, 계면활성제 공급.

섬유·석유화학

첨가제·처리액·공정 액체 연속 공급 라인.

Q 상담 전, 어떤 정보를 미리 알려주시면 도움이 되나요? +

모델명을 모르셔도 괜찮습니다. 아래 항목 중 알고 계신 것만 알려주셔도 제품군과 구성을 훨씬 빠르게 좁혀드릴 수 있습니다.

  • 액체 종류 — 물·배지·시약·용매·점성액·부식성·인화성 등
  • 유량 범위 — 예: 10 mL/min, 500 mL/min, 2 L/min
  • 사용 방식 — 연속 이송 / 반복 분주 / 미량 주입 / 대용량
  • 분주 조건 — 1회 분주량, 반복 횟수, 분주 간격
  • 사용 시간 — 단시간 사용 / 장시간 연속 운전
  • 채널 수 — 1·2·다중 + 채널별 독립 제어 필요 여부
  • 제어 방식 — 수동 / 외부 신호 / PLC·장비 연동
  • 설치 환경 — 실험실 / 생산라인 / 장비 내장 / 방폭 구역
  • 압력 조건 — 자연압·배관 저항·토출/흡입 높이
  • 튜브 조건 — 번호·내경·외경·재질·멸균·교체 주기
  • 정확도 요구 — 단순 이송 / 정량 분주 보정 필요
  • 필요한 구성 — 본체·헤드·튜브·액세서리·예비 소모품
