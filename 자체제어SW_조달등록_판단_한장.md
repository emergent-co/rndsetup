# 자체 제어SW 제품화 · 조달 등록 — 판단 (한 장)

작성 2026-07-21 · 근거: 벤처나라/나라장터 · GS인증(TTA·KTL) · GitHub 오픈소스 3갈래 병렬 조사

---

## 결론 (한 줄)

**지금 조달 등록·GS인증에 돈·시간 넣지 마라.** 조달몰은 영업이 끝난 뒤의 결제 레일이지 손님을 데려오는 채널이 아니다. 순서를 지키면 그냥 행정, 뒤집으면 전액 매몰비용.

---

## 확정된 사실 (믿어도 됨)

- **벤처나라·종합쇼핑몰은 검색·발견형 마켓이 아니다.** 영업으로 수요기관을 만든 뒤, 그 기관이 조달 절차로 결제할 때 쓰는 레일이다. "등록해두면 팔린다"는 거짓.
- **등록 순서**: 물품목록화(세부품명 → 물품식별번호) → 계약(MAS/3자단가) → 몰 상품등록. 계약 없는 물건은 몰에 안 올라간다.
- **수입 펌프(리드플루이드·Alicat)는 벤처나라 부적합** — 직접생산·국내OEM이 아님. 벤처나라에 올릴 수 있는 건 **자체 개발 SW 뿐**.
- **GS인증**: 자체 상용SW는 1등급 필수. 비용 700만~1,500만원(TTA 중소 30%할인 최저 ~700만), 대기 포함 3~6개월+, 버전 변경 시 재인증. 종합쇼핑몰 3자단가의 사실상 관문. 부산은 동의대 CIDI(인증기관)+부산정보산업진흥원 지원사업으로 비용 절감 가능.
- **시장 실체**: 조달에서 실제 팔리는 과학용SW는 대부분 해석·시뮬레이션 라이선스. 하드웨어 제어·로깅 SW를 단품 카탈로그로 반복판매하는 시장은 얇다(보통 장비 번들 or 용역 납품).

## 기술적 빈틈 (실재함)

오픈소스에도 상용에도 없는 조합:
**리드플루이드 연동펌프 + 시린지 + Alicat MFC를 한 화면 + 유량 스케줄/램프 + 다펌프 동기 + CSV 로깅 + 클릭식 GUI + 한국어/국내 A/S.**

- 오픈소스는 드라이버(통신)까지만. 완성 앱 Poseidon(★231)은 자작 하드웨어 전용·2019년 정체, flowchem(★22)은 고가 장비용·코드 편집 방식. 한국어 전무.
- 상용(Harvard·CETONI·Watson-Marlow)은 전부 자사 장비 락인·영어·고가.

→ 빈틈은 진짜다. **단 통신 계층으로는 못 판다(무료로 이미 풀림).** 팔리는 지점은 완성도 + 이종장비 통합 + 한국어/A/S.

## 하지 마라

- 영업·수요 미검증 상태에서 GS인증 선투자.
- 조달 등록부터 하고 손님 기다리기.
- "코딩 잘하는 연구자"를 타깃으로 잡기 — 무료 오픈소스로 자체 구축하고 이탈함.
- 검색량·SEO로 이 SW 수요를 검증하기 — 이 니치엔 검색 자체가 없어서 결과가 무의미.

---

## 해라 (순서 고정)

1. **기존 펌프 고객에게 직접 확인** — 다대수 동기·유량 스케줄·PC 로깅·번들SW 한글화 요구가 반복되는가. **이 답이 No면 여기서 중단.**
2. Yes면 SW를 단품이 아니라 **"장비 세트 + 제어SW 번들"**로 끼워 판다. 타깃 = 코딩 원치 않는 산업/QC/소규모 랩.
3. 그 고객이 조달로 사겠다고 하면 **그때** 물품목록화 — 응용과학용소프트웨어(4323260501) 해당 여부부터 조달청 물품목록과에 타진.
4. **반복 수요·공공 납품이 가시화된 뒤에만** GS인증 1등급(부산 CIDI/BIPA 지원 끼고).
