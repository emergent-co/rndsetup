기체 유량 제어 —
질량유량계(MFC) 셋업 정보

질량유량계(MFC)는 기체 유량을 전자식으로 정밀 제어하는 장치입니다. 로타미터와의 차이, 튜브퍼니스·CVD·촉매 반응에서의 쓰임, 선택 기준과 다중 MFC 동기 제어를 정리했습니다.

질량유량계(MFC, Mass Flow Controller)는 기체의 질량 유량을 센서로 측정하고 내장 밸브로 설정값에 맞춰 자동 조절하는 전자식 유량 제어 장치입니다. 온도·압력이 변해도 질량 기준으로 유량을 일정하게 유지해 재현성 있는 가스 분위기를 만듭니다.

MFC와 로타미터, 무엇이 다른가

로타미터(가변면적 유량계)는 관 속 부표의 위치를 눈으로 읽고 손으로 밸브를 여닫아 유량을 맞춥니다. 간단하지만 온도·압력·가스 종류에 따라 지시값이 달라지고, 값이 사람 손과 눈에 의존합니다. 반면 MFC는 질량 유량을 센서로 측정하고 밸브를 전자적으로 제어하므로, 설정값을 입력하면 조건이 변해도 같은 질량 유량을 재현성 있게 유지합니다.

  • 측정 기준 — 로타미터는 체적 유량(조건 의존), MFC는 질량 유량(조건 무관 재현).
  • 제어 방식 — 로타미터는 수동, MFC는 설정값 기반 자동 제어.
  • 기록·연동 — MFC는 통신으로 유량을 읽고 기록하며 소프트웨어로 스케줄 제어가 가능.

언제 쓰나

  • 튜브퍼니스 가스 분위기 제어 — N2·Ar·H2·공기 등의 유량과 혼합비를 재현성 있게 유지.
  • CVD(화학기상증착) — 전구체·캐리어 가스 유량을 정밀 제어해 증착 조건을 일정하게.
  • 촉매 반응 — 반응 가스 유량·비율을 정확히 잡아 활성·선택도 실험의 조건을 통제.
  • 가스 혼합비 제어 — 여러 가스를 정해진 비율로 섞어 목표 조성을 만듦.

Python 예제 — MFC로 가스 유량·혼합비를 코드로 제어하려면?

RS-485(Modbus)를 지원하는 MFC라면 가스별 유량 setpoint를 써서 혼합비를 재현합니다. (정확한 레지스터·명령은 MFC 매뉴얼을 따르세요.)

import minimalmodbus, time
# 레지스터 주소는 MFC 매뉴얼(Register Map) 참조
REG_SP = 0x0000                              # ← MFC 유량 setpoint 레지스터(매뉴얼 값)

def mfc(addr):
    m = minimalmodbus.Instrument('COM3', addr)   # 슬레이브 주소(1~247)
    m.serial.baudrate = 9600
    m.serial.parity = minimalmodbus.serial.PARITY_EVEN   # 8E1
    return m

n2, ar = mfc(1), mfc(2)          # N2, Ar 라인
n2.write_float(REG_SP, 200.0)    # N2 200 sccm
ar.write_float(REG_SP, 50.0)     # Ar 50 sccm  → 혼합비 4:1
time.sleep(3600)                 # 1시간 유지
