Cellab 관리자 — 통합 카탈로그 대시보드

전 카테고리 제품을 한 시트에서 편집. 카테고리별로 스펙 라벨이 다릅니다 (펌프=시리즈/유량/헤드/제어, 튜브=재질/내경/외경/길이, 시린지=용량/재질/타입/용도, 펌프헤드=시리즈/채널/유량/호환튜브, 피팅=종류/사이즈/재질/용도).
GitHub 자동 저장 (cellab.kr 전용)
1회 셋업: GitHub PAT 발급 → Repository access: emergent-co/cellab만 · Permissions: Contents (Read & Write)
저장 위치: 사장님 브라우저 localStorage. 다른 사람·서버 접근 불가. 토큰 만료 시 다시 발급해서 갱신.
작업 흐름
1
현재 데이터 불러오기로 _build/products.json 로드 또는 엑셀 가져오기로 .xlsx 업로드.
2
시트에서 카테고리 드롭다운으로 한 종류씩 보면서 편집. 행 추가 시 카테고리 먼저 선택하면 컬럼 의미가 명확. 가격은 숫자만 (콤마/원 없이).
3
JSON 다운로드 → 받은 products.json을 _build/products.json에 덮어쓰기 → build.bat 실행 → 6개 카테고리 페이지 자동 갱신.
데이터 입출력
엑셀 첫 행은 헤더. 컬럼명: 카테고리 / id / 모델명 / 코드 / 스펙1 / 스펙2 / 스펙3 / 스펙4 / 가격 / 카탈로그P / 키워드1·2·3
카테고리 id 모델명 코드 스펙1 스펙2 스펙3 스펙4 가격(원) 카탈로그P 키워드1 키워드2 키워드3 이미지URL
