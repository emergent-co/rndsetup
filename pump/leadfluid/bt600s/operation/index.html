Lead Fluid BT600S 설치·조작 영상 기준

Lead Fluid BT600S 연동펌프를 처음 쓸 때 알아야 할 것을 한 번에 정리했습니다. 본체에 헤드를 달고 튜브를 끼우는 설치부터, 키로 돌리는 기본 조작, 속도를 세밀하게 맞추는 속도 세팅까지. 준비물은 드라이버 하나면 충분합니다.

이 글의 순서
1. 설치 — 헤드 장착·튜브 끼우기
2. 기본 조작 — 내부 제어 모드
3. 속도 세팅 — 세밀 조절·원클릭 최고/최저

1. 설치 — 펌프헤드 장착과 튜브 끼우기

개봉하고 구성품부터 확인해요

패킹 리스트를 보면서 빠지거나 손상된 게 없는지 확인합니다. 기본 구성은 드라이버(본체), YZ15T 펌프헤드, 제품 카탈로그, 매뉴얼, 전원 코드, 부속품입니다.

설치 공간 — 본체 뒷면에 최소 200mm의 여유 공간을 둡니다. 방열과 케이블 연결을 위한 공간이라, 벽에 너무 붙이면 나중에 번거로워질 수 있습니다.

YZ15T 펌프헤드 장착하기

헤드 장착은 축 결합 → 나사 구멍 정렬 → 고정 나사 체결의 세 동작입니다. 먼저 펌프헤드의 평평한 축을 드라이버 커플링의 홈 방향에 맞춰 밀어 넣습니다. 축 단면이 한쪽이 평평한 모양이라 방향이 맞아야 끝까지 들어갑니다.

YZ15T 헤드의 평평한 축을 커플링 홈에 맞춰 밀어 넣기

축이 들어갔으면 헤드를 돌려 헤드와 드라이버 브래킷의 나사 구멍을 맞추고, 고정 나사 2개를 고정홀에 넣어 조입니다.

나사 구멍을 맞춰 고정 나사 2개로 헤드를 단단히 고정

튜브 끼우기

헤드 위쪽의 레버를 당겨 헤드를 엽니다. 튜브를 헤드 안에 부드럽게 넣고 꼬임 없이 곧게 편 뒤, 레버를 반대 방향(수평 위치)으로 돌려 닫으면 튜브 장착이 끝납니다.

레버를 열어 튜브를 넣고, 레버를 수평으로 돌려 닫으면 설치 완료

2. 기본 조작 — 내부 제어 모드

헤드와 튜브를 장착했으면 실제로 펌프를 돌릴 차례입니다. 가장 기본이 되는 조작이 내부 제어 모드(internal control mode)입니다. 앞면 패널의 키만으로 펌프를 직접 제어하는 방식이라, PC나 외부 신호 없이도 바로 쓸 수 있습니다.

전원 켜고 모드 전환

뒷면 전원 스위치를 켜면 LCD에 환영 메시지가 잠깐 뜬 뒤 메인 화면으로 들어갑니다. 여기서 MODE 키를 눌러 내부 제어 모드로 맞춥니다.

MODE 키로 내부 제어 모드 전환 (화면에 Int 표시)

속도·방향 설정과 운전

up / down 키로 원하는 속도(rpm)를 맞춥니다. 화면 아래쪽에 현재 속도가 rpm으로 표시됩니다. 회전 방향은 direction(정역) 키로 바꾸고, start/stop 키로 펌프를 시작하거나 멈춥니다.

up/down 키로 설정할 속도를 조절 — 화면에 Int와 rpm이 함께 표시

최대 속도로 바로 돌리기

full speed 키를 누르면 펌프가 최대 속도로 돌아갑니다. 세척이나 빠른 배출처럼 잠깐 최고 속도가 필요할 때 편리합니다.

full speed 키를 누르면 펌프가 최대 속도로 운전

3. 속도 세팅 — 세밀 조절과 원클릭 최고·최저

BT600S는 속도를 아주 세밀하게 조절할 수 있고, 버튼 조합 하나로 최고 속도나 최저 속도로 바로 점프할 수도 있습니다.

속도 분해능 — 구간에 따라 달라져요

메인 화면에서 up/down 키로 속도를 조절할 때, 속도 구간에 따라 조절 단위(분해능)가 달라집니다. 0.1 ~ 100 rpm 구간은 0.1 단위로 세밀하게, 100 ~ 600 rpm 구간은 1 단위로 조절됩니다. 저속에서 촘촘하게 맞출 수 있어 정밀한 유량 세팅에 유리합니다.

저속 구간에서는 0.1 단위(예: 90.4rpm)로 세밀하게 조절

빠르게 올리고 내리기

up 또는 down 키를 길게 누르고 있으면 속도가 연속으로 빠르게 오르내립니다. 큰 폭으로 바꿀 때 하나씩 누르지 않아도 됩니다.

원클릭으로 최고·최저 속도로

여기가 핵심입니다. full speed 키를 누른 상태에서 up 키를 누르면 속도가 곧바로 최고로, full speed 키를 누른 상태에서 down 키를 누르면 곧바로 최저로 설정됩니다.

full speed + up은 최고, full speed + down은 최저 속도로 한 번에 설정

예를 들어 full speed + up을 쓰면 화면 속도가 곧장 최고값(600 rpm)으로 올라갑니다.

full speed + up으로 최고 속도 600rpm이 한 번에 설정된 모습

정리

  • 설치 — 뒤 200mm 공간 확보 → 헤드 축 결합 → 나사 구멍 정렬 → 고정 나사 2개 체결 → 레버 열고 튜브 넣고 닫기
  • 기본 조작 — 전원 ON → MODE 키로 내부 제어 모드 → up/down 속도, direction 방향, start/stop 운전 → full speed 최대 속도
  • 속도 세팅 — 0.1~100은 0.1 단위·100~600은 1 단위, 길게 눌러 연속 증감, full speed+up/down으로 원클릭 최고·최저
