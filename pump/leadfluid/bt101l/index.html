리드플루이드 BT101L 연동펌프 (LeadFluid BT101L)
— RS485로 PC 제어·도금 현장 검증

BT101L은 리드플루이드(LeadFluid) 지능형 L 시리즈 연동펌프로, 유량 0.00011~750 mL/min, RS485·Modbus RTU로 PC 원격 제어됩니다. 실험셋업연구소는 실제 도금 도입사례와 제어 소프트웨어·국내 직접 A/S를 함께 제공합니다.
유량 범위0.00011 ~ 750 mL/min (펌프헤드·튜브 조합에 따름)
통신RS485 · Modbus RTU (9600 · 8E1 · 슬레이브 1~247, 사양서 기준)
시리즈L (지능형 — 디스플레이 유량·회전수·정역 설정)
대표 용도연구실 분주·샘플링·연속 이송, 코팅·도금액 이송

BT101L은 어떤 펌프인가?

리드플루이드(LeadFluid) L 시리즈에 속하는 지능형 연동(페리스탈틱) 펌프로, 디스플레이에서 유량·회전수·정역회전을 직접 설정하는 실험실·소량 생산용 모델입니다. 튜브만 교체하면 유체가 펌프 본체에 닿지 않아 부식성·점성 유체나 오염에 민감한 코팅·도금 용액 이송에 적합합니다. 정확한 유량은 펌프헤드·튜브 규격 조합에 따르며, 용도별 선정은 펌프 고르는 방법이나 문의로 안내합니다.

PC(RS485·Modbus)로 제어되나?

됩니다. BT101L은 RS485·Modbus RTU를 지원하며 9600·8E1, 슬레이브 주소 1~247 범위에서 통신합니다(사양서 기준). 실험셋업연구소에서 구매하시면 PC 제어 소프트웨어를 무상 제공해 유량·구동·정지를 자동 스케줄로 운전하고 운전 로그(CSV)로 재현할 수 있습니다. 내 펌프가 제어되는지는 PC 제어 확인에서, 자동화 전체는 소프트웨어 제어에서 볼 수 있습니다.

Python 예제 — BT101L을 코드로 제어하려면?

RS485·Modbus RTU를 지원하므로 범용 라이브러리로 유량·구동을 제어합니다. (정확한 레지스터 주소·바이트오더는 BT101L 매뉴얼을 따르세요.)

import minimalmodbus, time
# 레지스터 주소·바이트오더는 BT101L 매뉴얼(Register Map) 참조
REG_FLOW, REG_RUN = 0x0000, 0x0000        # ← 매뉴얼 값으로 대체

pump = minimalmodbus.Instrument('COM3', 1)   # 포트, 슬레이브 주소(1~247)
pump.serial.baudrate = 9600
pump.serial.parity = minimalmodbus.serial.PARITY_EVEN   # 8E1

pump.write_register(REG_RUN, 1, functioncode=6)   # Start
pump.write_float(REG_FLOW, 12.5)                  # 유량 12.5 mL/min
time.sleep(600)                                   # 10분 유지
pump.write_register(REG_RUN, 0, functioncode=6)   # Stop

실제 어떤 현장에 쓰였나?

도금 도입사례 — 실험셋업연구소가 구성한 도금 라인 셋업(BT101L 2대·RS485 제어·저울 실측 유량 캘리브레이션): 도금 다펌프 제어 사례. 경쟁사가 스펙만 나열할 때, 우리는 "이 모델로 이런 실험을 이렇게 돌렸다"를 셋업 단위로 공개합니다.

정품·A/S·구매는?

실험셋업연구소는 리드플루이드 정품을 공급하고 국내에서 직접 A/S합니다. 구매 시 국내 직접 A/S·제어 소프트웨어·3년 무상보증(구매 조건)이 적용되고, 타사 구매품도 무료 진단 후 수리합니다. 인증서·수리 사례는 국내 A/S·정품·보증에서 확인하세요.
