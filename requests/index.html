프로그래밍 제어

Modbus·RS-485·Python으로 펌프를 제어합니다

손으로 버튼을 누르지 않고 PC·스크립트로 펌프를 움직입니다. LeadFluid 정량·연동펌프는 RS-485(Modbus RTU) 인터페이스를 제공하며, 셀렙 제어 소프트웨어가 유량·RPM·회전 방향·기동/정지를 명령으로 주고받습니다.

직접 코드로 통합하고 싶다면 Modbus 레지스터 맵과 Python 예제를 제공합니다. 센서를 읽어 유량을 조정하는 폐루프, 다른 장비와의 시퀀스 연동 등 기존 실험 자동화 스크립트에 펌프를 그대로 넣을 수 있습니다.

유량 스케줄 · 레시피

시간에 따라 유량을 바꾸고, 레시피로 저장·반복합니다

일정 시간 특정 유량 유지, 선형 증감(ramp), 단계별 스텝을 시퀀스로 짜고 저장했다가 그대로 반복 실행합니다. "언제 얼마를 흘릴지"가 실험 조건인 경우에 특히 유용합니다.

  • 관류 배지 교환 스케줄 — 정해진 주기로 유량 전환
  • 농도 구배·조건 스윕 — 유량을 단계적으로 변경
  • 레시피 파일 저장 — 실험 간 동일 조건 재사용

다펌프 동기 · 무인 운전

여러 펌프를 동기화하고, 며칠~몇 주를 무인으로 돌립니다

여러 펌프를 한 화면에서 개별·동기 제어합니다. 조건군마다 다른 유량을 채널별로 독립 설정하거나, 여러 펌프를 같은 타임라인에 묶어 동시에 기동합니다.

한 번 시퀀스를 걸면 사람이 없어도 설정대로 유량을 유지해 장시간 무인 관류·연속배양을 흔들림 없이 이어갑니다. 대조군을 한 번에 돌리고, 밤사이 데이터 손실 없이 재현 데이터를 쌓습니다.

기록 · 재현

운전 로그를 남기고 같은 운전을 재현합니다

무엇을 언제 얼마나 흘렸는지 유량·누적량·상태 변화를 타임스탬프와 함께 기록하고 CSV로 내보내, 데이터 분석·논문 부록에 그대로 씁니다.

같은 프로파일을 다시 불러 재실행하면 이전과 동일한 운전을 재현할 수 있어, 재현성이 성립 조건인 연구에서 실험 간 편차를 줄입니다. (정밀 유량 피드백이 필요하면 Alicat 질량유량계(MFC) 연동 폐루프 제어도 별도 지원.)
