프로그래밍 제어 — Modbus·RS-485·Python으로 펌프를 제어합니다

손으로 버튼을 누르지 않고 PC·스크립트로 펌프를 움직입니다. LeadFluid 정량·연동펌프는 RS-485(Modbus RTU) 인터페이스를 제공하며, 실험셋업연구소 제어 소프트웨어가 유량·RPM·회전 방향·기동/정지를 명령으로 주고받습니다.

직접 코드로 통합하고 싶다면 Modbus 레지스터 맵과 Python 예제를 제공합니다. 센서를 읽어 유량을 조정하는 폐루프, 다른 장비와의 시퀀스 연동 등 기존 실험 자동화 스크립트에 펌프를 그대로 넣을 수 있습니다.

import minimalmodbus
p = minimalmodbus.Instrument('COM3', 1)
p.serial.baudrate = 9600
p.serial.parity = minimalmodbus.serial.PARITY_EVEN   # 8E1
CDAB = minimalmodbus.BYTEORDER_LITTLE_SWAP
p.write_register(4022, 2, functioncode=6)   # 단위 ml/min
p.write_float(4015, 12.5, byteorder=CDAB)   # 유량 12.5
p.write_register(4126, 1, functioncode=6)   # Start (BT101L)

다펌프 동기 · 무인 운전여러 펌프를 동기화하고, 며칠~몇 주를 무인으로 돌립니다

여러 펌프를 한 화면에서 개별·동기 제어합니다. 조건군마다 다른 유량을 채널별로 독립 설정하거나, 여러 펌프를 같은 타임라인에 묶어 동시에 기동합니다.

한 번 시퀀스를 걸면 사람이 없어도 설정대로 유량을 유지해 장시간 무인 관류·연속배양을 흔들림 없이 이어갑니다. 대조군을 한 번에 돌리고, 밤사이 데이터 손실 없이 재현 데이터를 쌓습니다.
